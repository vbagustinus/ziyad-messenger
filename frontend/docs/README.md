# Frontend Documentation

Dokumentasi untuk platform frontend admin dashboard.

## Tujuan Platform

Menyediakan panel kontrol admin untuk operasi harian sistem komunikasi enterprise berbasis LAN.

## Peran Utama

- Login admin dan manajemen sesi.
- Pengelolaan user, role, department, channel, dan device.
- Monitoring kondisi sistem dan visibilitas audit.

## Skill Inti

- Next.js (App Router) + TypeScript.
- State management dengan Zustand.
- API integration dengan Axios.
- Data visualization untuk operational insight.
