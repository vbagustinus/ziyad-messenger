# Client Documentation

Dokumentasi untuk platform client (desktop/mobile).

## Tujuan Platform

Memberikan pengalaman komunikasi realtime untuk end-user pada lingkungan LAN enterprise.

## Peran Utama

- Autentikasi user ke backend.
- Kirim/terima pesan realtime dan akses riwayat.
- Menampilkan status presence dan alur file transfer.

## Skill Inti

- Flutter cross-platform client development.
- WebSocket + HTTP fallback untuk transport pesan.
- Local persistence untuk token/session.
- Integrasi endpoint lintas lingkungan LAN.
